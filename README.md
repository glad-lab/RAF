## Run

### Optimization
We've prepared scripts to quickly run the code. Simply run the script like:
```bash
bash run_raf.sh
```

### Evaluation

The evaluation script helps to evaluate adversarial prompts generated by different approaches:

```bash
bash evaluate.sh
```
